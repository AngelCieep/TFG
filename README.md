# React Visual Builder

## Descripción

Herramienta que permite diseñar interfaces web mediante un editor visual y generar automáticamente un proyecto React a partir de un archivo JSON.

El sistema separa el diseño de la implementación utilizando un formato intermedio que describe la estructura de la interfaz.

## Tecnologías

* React
* Node.js
* JavaScript (ES Modules)
* CSS Grid
* JSON
